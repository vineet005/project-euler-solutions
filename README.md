# Project Euler Solutions (Java)

Optimized Java solutions to Project Euler problems.

This repository contains my curated and well‑structured Java solutions to selected Project Euler problems. All solutions are stored inside a single `src` folder, and each file includes a `main` method so it can be run independently. Every solution is written with clean code, mathematical reasoning, and optimized logic.

---

## 📂 Project Structure
src/
├── Problem_1_Difficulty_0.java
├── Problem_2_Difficulty_0.java
├── Problem_23_Difficulty_1.java
├── Problem_66_Difficulty_2.java
└── ...

Each class follows the naming format: Problem_<number>Difficulty<level>.java
Example: Problem_1_Difficulty_0.java


---

## 📝 Commit Message Format

Every commit follows this structure: Problem <number> - Difficulty <level> - <Problem Name>
Example: Problem 1 - Difficulty 0 - Multiples of 3 or 5



This keeps the history clean, searchable, and consistent.

---

## 🎯 Difficulty Levels

- **Difficulty 0** — Basic problems using simple math or loops  
- **Difficulty 1** — Intermediate problems involving number theory or combinatorics  
- **Difficulty 2** — Advanced problems requiring optimized algorithms  
- **Difficulty 3** — Very hard problems involving deep math or heavy optimization
